# LINE 記帳小工具

讓親友只要有 LINE 跟一個 Google 帳號,就能零負擔地記錄個股交易明細、追蹤損益，不需要學任何記帳 App,打字、傳照片(後續階段)或直接在自己的 Google 試算表填資料都能記帳。

> 🚧 **目前狀態:Phase 1 MVP 實作中（~88%）。** 核心記帳、損益引擎、OAuth 連結、查詢/撤銷/同步功能已完成並部署至 Cloud Run；LIFF OAuth 手機端驗證進行中。

---

## 為什麼做這個

身邊親友想開始記錄股票交易,但市面上看盤軟體、券商 App 已經很多了,為什麼還要自己做一個?因為這個工具想解決的其實是不同的問題:

- **看盤 App 解決「即時報價」,沒有解決「多人、多帳戶、長期記帳」**:券商 App 看得到自己在那家券商的庫存損益,但看不到「我在三家券商加起來總共持有多少」,更不可能讓家人朋友的持股彙整在同一個視角下比較。
- **大家共用一份 Google 試算表能解決協作,但解決不了隱私**:最直覺的做法是大家共用一份表格,但這樣等於把所有人的財務細節攤在同一份檔案裡,而且資料通常會落到某一個人(管理者)手上集中保管。
- **市面上的記帳 App 要大家各自下載、註冊、學新介面**:對「只是想幫家人朋友架一個簡單工具」這種場景太重了。

這個專案想解決的核心問題其實是:

> **怎麼讓多個人共用同一套記帳工具,又不需要任何人(包含開發者自己)集中保管所有人的財務資料?**

整個架構的核心設計都是繞著這個隱私問題在轉,細節見下方「核心設計理念」。

## 專案亮點

- **零信任管理員的多人協作架構**:每位親友的試算表活在他自己的 Google Drive,不是集中存在開發者的雲端空間——這是整個專案技術上最核心的挑戰,也是跟一般「大家共用一份雲端表格」做法最大的不同。
- **跨帳戶、跨券商的彙整視角**:券商 App 只能看到自己這家的庫存,這個工具讓親友可以把多個帳戶(不同券商、不同身份)彙整在同一套損益邏輯下檢視。
- **零設定、零新介面**:不需要下載新 App、不需要註冊新帳號——用大家已經在用的 LINE 聊天記帳,資料本身又是大家熟悉的 Google 試算表格式,隨時可以打開來看、自己編輯。
- **完全用免費額度撐起來的工程取捨**:Cloud Run 免費額度、Cloud Scheduler 免費額度、Firestore 免費額度、Google 官方 API 免費方案,刻意限制在零成本的條件下做架構設計,逼自己想清楚「免費方案的限制是什麼、怎麼繞過去」,而不是遇到限制就直接砸錢解決。
- **貼著台股零股交易實際情況設計的記帳邏輯**:換行批次記帳、中文股名模糊比對、移動加權平均成本法、已實現/未實現損益分流、除權息與配股調整——這些不是看盤軟體的強項,是這個工具真正想做好的部分。

## 核心設計理念

- **隱私優先**:開發者(管理員)不持有、不集中保管任何親友的財務資料。每位親友透過自己的 Google 帳號 OAuth 授權,試算表建立在**親友自己的 Google Drive** 裡,不是集中存在管理員的雲端空間。後端只存一張極小的「連結關係」對照表(LINE 帳號 ↔ 試算表 ID ↔ 加密授權憑證),不存任何實際記帳資料。
- **雙入口都要通**:親友可以用 LINE 文字記帳,也可以直接打開自己的 Google 試算表手動輸入,兩條路殊途同歸,資料永遠以試算表為準。
- **先求 MVP,再求完整**:核心記帳與損益邏輯優先上線,手續費精算、照片 AI 辨識等進階功能規劃在後續階段,不卡住第一版上線時程。
- **溫度感**:這是給親友用的工具,不是給陌生大眾的 SaaS,語氣與互動設計刻意避免冷冰冰的系統感。

## 技術架構

| 項目 | 選擇 |
|---|---|
| 後端 | Python 3.10+ / FastAPI(大量 I/O 等待操作,適合 async) |
| 資料儲存 | Google Sheets(親友自己 Drive 裡的試算表)+ OAuth 2.0 |
| 系統對照表 | Google Cloud Firestore(僅存連結關係,不存記帳資料) |
| 部署平台 | Google Cloud Run + Cloud Scheduler(免費額度) |
| 前端 | LINE LIFF(手機)+ RWD 網頁(電腦自動升級寬幅版) |
| 外部資料源 | 臺灣證券交易所 OpenAPI、證券櫃買中心 OpenAPI |

## 目前確定開發的功能範圍(MVP,Phase 1)

- **LINE 記帳**:文字記帳(買/賣/股息/配股),支援換行批次多筆、股票代碼或中文名稱模糊比對、僅限 1:1 私訊(不處理群組訊息,保護隱私)
- **試算表記帳**:親友可直接在自己的試算表手動輸入,後端定期同步讀回計算
- **損益引擎**:移動加權平均成本法計算,已實現/未實現損益分流,賣超防呆,現金股息與股票股利(配股)都支援
- **LINE 直接查詢**:選單按鈕查看目前庫存、損益、總資產
- **基本網頁(LIFF)**:登入連結狀態、庫存列表、簡單損益顯示
- **帳號安全**:OAuth 授權加密儲存、失效自動偵測與主動通知重新連結、LIFF 身分驗證採 `id_token`(不依賴網址參數)
- **防呆機制**:刪除上一筆(限時)、批次部分失敗時僅提示有問題的行、輸入不清楚時引導重新輸入
